== Description ==

Hit Counter Ultimate is a simple wordpress plugin that embeds a rotating counter in your blog and shows the number of visits your website has.
By default the web counter will be displayed in your footer, to change the location of the counter simply navigate to your widgets page and embed it anywhere you want in your blog.
You can also change the hit counter style so it can fit with the design of your blog. Navigate to -> Settings -> Hit Counter Ultimate and select your preffered style.


= Plugin Settings =

* Count unique visitors - checkbox
* Exclude ip addresses - one per line
* Display in footer - checkbox
* Counter Alignment - left,center,right
* Set hit counter to - a specified number
* Credits - enable or disable
* Check for updates automatically - enable or disable

== Installation ==

This section describes how to install the plugin and get it working.

1. Unzip the downloaded archive
2. Extract **hit-counter-ultimate.zip** and upload the folder **hit counter ultimate** to */wp-content/plugins/* directory
3. Activate the plugin from *Plugins > Hit Counter Ultimate > Activate*, under WordPress admin interface
4. In the *Appearance > Widgets* menu, drag to your sidebar the Hit Counter Ultimate Plugin.
5. Change settings or manage the plugin from *Settings > Hit Counter Ultimate*.
6. More information about plugin at developer page [Hit Counter Ultimate](http://www.packages-seo.com/hit-counter-ultimate).
